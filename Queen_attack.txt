#!/usr/bin/env bash

Declare row and columns
A chessboard can be represented by an 8 by 8 array.
Horizontal side [a,b,c,d,e,f,g,h]
Column [0,1,2,3,4,5,6,7]

Two queen are represented by w and b 
inputs will decide if a queen can attack other queen

In the game of chess, a queen can attack pieces which are on the same row, column, or diagonal.

positions are input as -w $1,$2 -b $3,$4  
$1 column $2 row $3 column $4 row

so, calculate both positions on 8x8 array and see if they are on same row column or inj diagnoal position to each other and can attack each other.
